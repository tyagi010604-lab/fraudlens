# FraudLens – Power BI Dashboard Setup Guide
## File: powerbi/PowerBI_Setup_Guide.md

This guide explains how to build the FraudLens Power BI dashboard using the
`data/processed_jobs.csv` output file generated by the project.

---

## STEP 1: Open Power BI Desktop
Download free from: https://powerbi.microsoft.com/desktop

---

## STEP 2: Import the Data

1. Click **Home → Get Data → Text/CSV**
2. Browse to `data/processed_jobs.csv`
3. Click **Load**

---

## STEP 3: Data Transformations (Power Query)

Click **Transform Data**, then:

### Add Risk Color Column
```
= Table.AddColumn(#"Previous Step", "Risk Color",
    each if [risk_level] = "High Risk" then "#e74c3c"
    else if [risk_level] = "Medium Risk" then "#f39c12"
    else "#27ae60")
```

### Add Fraud Label Column
```
= Table.AddColumn(#"Previous Step", "Fraud Label",
    each if [fraudulent] = 1 then "Fake" else "Real")
```

Click **Close & Apply**.

---

## STEP 4: Create Measures (DAX)

In the **Model** view, create these measures:

```dax
Total Jobs = COUNTROWS(job_postings)

Fake Jobs = CALCULATE(COUNTROWS(job_postings), job_postings[fraudulent] = 1)

Real Jobs = CALCULATE(COUNTROWS(job_postings), job_postings[fraudulent] = 0)

Fraud Rate % = DIVIDE([Fake Jobs], [Total Jobs]) * 100

Avg Trust Score = AVERAGE(job_postings[trust_score])

Avg Trust Real = CALCULATE(AVERAGE(job_postings[trust_score]), job_postings[fraudulent] = 0)

Avg Trust Fake = CALCULATE(AVERAGE(job_postings[trust_score]), job_postings[fraudulent] = 1)

High Risk Count = CALCULATE(COUNTROWS(job_postings), job_postings[risk_level] = "High Risk")
```

---

## STEP 5: Build the Dashboard (4 Report Pages)

### PAGE 1 – Executive Summary
| Visual | Fields |
|--------|--------|
| Card   | Total Jobs |
| Card   | Fake Jobs |
| Card   | Fraud Rate % |
| Card   | Avg Trust Score |
| Donut Chart | Fraud Label (legend), count (values) |
| Donut Chart | Risk Level (legend), count (values) |

### PAGE 2 – Fraud Distribution
| Visual | Fields |
|--------|--------|
| Clustered Bar | employment_type (axis), Fraud Label (legend), count |
| Stacked Column | location (axis), Fraud Label (legend), count |
| Matrix | risk_level rows, fraudulent column, count values |
| Slicer | employment_type |

### PAGE 3 – Trust Score Analysis
| Visual | Fields |
|--------|--------|
| Histogram (column chart) | trust_score (X), count (Y), colored by Fraud Label |
| Scatter Chart | trust_score (X), scam_keyword_count (Y), size=missing_fields |
| Line + Column | risk_level (axis), Avg Trust Score (line), Fake Jobs (column) |
| KPI Card | Avg Trust Fake vs Avg Trust Real |

### PAGE 4 – Salary & Risk Insights
| Visual | Fields |
|--------|--------|
| Column Chart | salary_anomaly (X), count (Y), Fraud Label (legend) |
| Treemap | industry (group), Fake Jobs (size) |
| Table | title, company, trust_score, risk_level (top 20 high risk) |
| Gauge | Avg Trust Score, min=0, max=100, target=70 |

---

## STEP 6: Styling Tips

- Set theme to **Accessible Default** (View → Themes)
- Use colors: Fake = `#e74c3c`, Real = `#27ae60`, Neutral = `#2980B9`
- Add a title text box on each page: **"FraudLens – [Page Name]"**
- Add your name and date in the footer

---

## STEP 7: Publish (Optional)
- Save as `FraudLens_Dashboard.pbix`
- Click **Publish** to Power BI Service (requires free account)

---

## Data Column Reference

| Column | Description |
|--------|-------------|
| job_id | Unique job ID |
| title | Job title |
| company_profile | Company name |
| fraudulent | 1=Fake, 0=Real |
| trust_score | 0–100 trust score |
| risk_level | Low/Medium/High Risk |
| missing_fields | Count of blank fields |
| scam_keyword_count | Suspicious keyword count |
| salary_anomaly | 1 if salary > 200K |
| has_company | 1 if company info present |
| employment_type | Full-time/Part-time/etc. |
| is_wfh | 1 if work-from-home |
